# 網頁版象棋 (Xiangqi Web) - v4.0

一個純前端開發的網頁版象棋遊戲，支援電腦對戰 (AI) 與雙人對戰。
作者：劉建良 @ 2025 (免費開源軟體)

## 🎮 功能特色 (v4.0 更新)

### 1. 智慧 AI 對戰
- **迭代加深引擎**: 採用 Minimax 搭配 Alpha-Beta 剪枝與迭代加深 (IDDFS) 技術，運算更深更廣。
- **位置價值評估**: 內建完整的「位置權重表 (PST)」，AI 懂得馬跳臥槽、車佔肋道等佈局觀念。
- **背景運算**: 使用 Web Worker 技術，在瀏覽器背景思考，確保介面流暢不卡頓。

### 2. 生動的遊戲體驗
- **動態音效**: 內建 Web Audio API 音效引擎，無需下載檔案即可發出落子、吃子、將軍、獲勝音效。
- **視覺輔助**:
    - **最後一步提示**: 自動高亮顯示對手剛走的棋路線。
    - **將軍特效**: 被將軍時，將/帥會閃爍紅色警示光芒。

### 3. 完整遊戲功能
- **雙模式**: 支援「人機對戰」與「雙人對戰 (Hotseat)」。
- **悔棋功能**: 支援無限悔棋，AI 模式下會自動退回兩步 (玩家回合)。
- **響應式設計**: 手機、平板、電腦皆能自動適配最佳版面。
